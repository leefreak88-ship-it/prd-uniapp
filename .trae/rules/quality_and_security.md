# 质量与安全 (Quality and Security)

## 具体约束条件
1.  **敏感信息保护**：
    *   **禁止**在前端代码中硬编码任何敏感信息，包括但不限于 API Keys、Secrets、Token、私钥等。
    *   敏感配置应通过环境变量或后端接口动态获取。
2.  **数据渲染优化**：
    *   `data` (Vue 2 Options API) 或响应式状态 (Vue 3 Composition API) 仅存储用于**视图渲染**的数据。
    *   与渲染无关的中间变量、临时状态或配置项，不应放入响应式系统，以减少 `setData` 通信开销及性能损耗。
3.  **注释规范**：
    *   关键业务逻辑、复杂算法、接口调用处必须添加详细的**中文注释**。
    *   组件 Props、Methods 需有对应的 JSDoc 或单行注释说明其用途。
